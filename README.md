## Installation

MXImageBrowser is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MXImageBrowser', :git => 'https://github.com/changbiao/MXImageBrowser.git'
```
